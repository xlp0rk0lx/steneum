What is Steneum?
----------------

Steneum is a lite version of Bitcoin using scrypt as a proof-of-work algorithm.
 - 5 minute block targets
 - subsidy halves in 300k blocks (~3 years)
 - 20 million total coins

The rest is the same as Bitcoin.
 - 25 coins per block
